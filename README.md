<h1> Atividade3-RP2040</h1>

<p>Este projeto implementa o controle de posição de um servo motor padrão (0° a 180°) utilizando um joystick analógico. 
Desenvolvido para a plataforma Raspberry Pi Pico em linguagem C, o sistema lê continuamente a posição do eixo horizontal do joystick 
e a converte em um sinal PWM.</p>
